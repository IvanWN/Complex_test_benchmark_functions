процедура AOA(число_агентов, макс_итер, пространство_поиска, функция_приспособленности)
   Инициализация популяции агентов со случайными позициями в пространстве_поиска
   Инициализация fitness каждого агента как бесконечность

   для каждого агента в популяции:
      приспособленность_агента = функция_приспособленности(агент)
      обновить fitness агента с приспособленность_агента

   для t от 0 до макс_итер:
      отношение_времени = t / макс_итер

      для каждого агента в популяции:
         новая_позиция = обновитьПозицию(агент, отношение_времени, пространство_поиска)
         новая_приспособленность = функция_приспособленности(новая_позиция)

         если новая_приспособленность < fitness агента:
            обновить позицию и fitness агента

   выбрать агента с наилучшей приспособленностью и вернуть его

конец процедуры

процедура обновитьПозицию(агент, отношение_времени, пространство_поиска)
   для каждой координаты в позиции агента:
      амплитуда = (верхний_предел - нижний_предел) / 2
      новая_координата = текущая_координата + амплитуда * sin(2 * π * отношение_времени)
      обеспечить, чтобы новая_координата находилась в пределах пространства_поиска

   вернуть обновленную позицию
